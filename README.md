# 👁️ Keen Eye Advertising

[![Vercel Deployment](https://img.shields.io/badge/Vercel-Deployment-black?style=flat&logo=vercel)](https://keeneye.gr/)
[![Vite](https://img.shields.io/badge/Vite-Project-646CFF?style=flat&logo=vite)](https://vitejs.dev/)

**Keen Eye Advertising** is a high-end web presence for a premier advertising agency based in Thessaloniki, Greece. Established in 1995, the agency specializes in high-fidelity signage, digital printing, and comprehensive branding solutions.

---

## ✨ Features

-   **Premium Editorial UI/UX**: A minimalist, sophisticated design system using custom typography (Libre Baskerville & Manrope) and a curated color palette.
-   **Dynamic Animations**: Smooth, hardware-accelerated entrance animations and scroll-triggered reveals for an engaging user experience.
-   **Service Portfolio**: Detailed showcase of core services including:
    -   **Signage (Επιγραφές)**: Custom signs for businesses.
    -   **Digital Printing**: High-quality ψηφιακές εκτυπώσεις.
    -   **Vehicle Wrapping**: Professional car and van graphics.
    -   **Branding**: Corporate identity and logo design.
-   **Mobile-First Performance**: Optimized for speed and responsiveness, ensuring a seamless experience across all devices.
-   **Optimized Assets**: Use of modern image formats (WebP/AVIF) and minified CSS for 100/100 Lighthouse performance scores.

---

## 🛠️ Tech Stack

-   **Frontend**: HTML5, Vanilla CSS (Modern CSS variables & Grid/Flexbox), JavaScript (ES6+)
-   **Build Tool**: [Vite](https://vitejs.dev/)
-   **Styling**: Consolidated Design System with custom animation frameworks.
-   **Performance**: Lighthouse-optimized critical path rendering.
-   **Deployment**: [Vercel](https://vercel.com/)

## 🚀 Getting Started

### Prerequisites

-   [Node.js](https://nodejs.org/) (v18 or higher)
-   [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/dervass/keeneye.git
   cd keeneye
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
